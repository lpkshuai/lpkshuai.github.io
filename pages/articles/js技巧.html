# 指数运算

计算n的m次方：

  1. ** 运算符
  2. << 运算符
  3. Math.pow()
// n**m
console.log(2**3); // 8
// n<<m-1
console.log(2<<3-1); // 8
// Math.pow(n, m)
console.log(Math.pow(2, 3)); // 8

数字转字符串：

  1. 连接一个空字符串
  2. toString() 方法
  3. String() 构造器
let num = 15;
console.log(num+'');
console.log(num.toString());
console.log(String(num));

字符串转数字：

    • 连接
  2. ~~ 连接
  3. parseInt() 方法
  4. Number() 构造器
let str = '15';
console.log(+str);
console.log(+true);
console.log(~~str);
console.log(~~true);
console.log(Number(str));
console.log(parseInt(str));
