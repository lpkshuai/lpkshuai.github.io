某次更新后运行项目 `npm run dev` 突然发现会在编辑器tab栏自动打开一个预览的窗口（此时编辑器内部和外部浏览器都会打开预览页面），由于平时习惯用外部浏览器预览项目，所以找到了关闭的设置。cursor的设置中找到相关设置关闭即可：
