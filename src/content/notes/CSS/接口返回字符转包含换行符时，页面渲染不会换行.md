### 描述：

接口返回数据字符串，使用插值语法 `{{ text }}` 渲染到页面时，内容的换行未生效。

### 解决：

使用 `white-space: pre-line;`

```jsx
<template>
  <div style="white-space: pre-line;">{{ text }}</div>
</template>

<script>
export default {
  data() {
    return {
      text: '第一卷\n经营商业企业之一般规则\n第一编\n商业企业主',
    };
  },
};
</script>
```
